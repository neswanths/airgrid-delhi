    Solution Tiers

    Three progressively deeper integration levels, each defensible on cost and impact.

    ⚡

    Quick Deploy

    Crowdsourced mobile sensing via public transport fleet integration. Retrofits existing DTC buses and auto-rickshaws with low-cost PM sensors.

    • No new infrastructure needed
    • Deploy in 2-4 weeks
    • ~500 mobile nodes possible
    Cost per sensor ₹2,000 – ₹5,000
    🚀

    Bold Expansion

    Automated drone survey corridors covering industrial belts and remote zones. ML-powered spatial interpolation fills data gaps.

    • Covers blind spots
    • On-demand hotspot surveys
    • ML interpolation engine
    Investment level Medium
    🏛️

    Long-lasting

    Permanent ward-level micro-station grid with a unified open API for third-party integration and academic research.

    • 272 permanent nodes
    • Open data API
    • 10+ year lifespan
    Cost per unit × 272 wards ₹15,000 / unit
